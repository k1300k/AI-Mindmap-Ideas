# IdeaFlow - 인터랙티브 마인드맵 대시보드

![Version](https://img.shields.io/badge/version-1.2.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)
![Development](https://img.shields.io/badge/dev-Vibe%20Coding-purple)
![Mobile](https://img.shields.io/badge/mobile-optimized-brightgreen)

**바이브코딩(Vibe Coding)** 프롬프트 방식으로 구성된 차세대 마인드맵 대시보드입니다. 
AI와의 대화를 통해 점진적으로 발전하며, **모바일 최적화**, **자동 저장**, **줌/패닝** 기능이 있는 아름답고 직관적인 인터페이스를 제공합니다.

## 📌 프로젝트 소개

**"질문으로 접근하는 생활형 불편 대응 아이디어"**를 시각화하고 관리하기 위한 마인드맵 도구입니다. 
일상 생활에서 발견한 불편함을 체계적으로 정리하고, 해결 아이디어를 확장해 나갈 수 있습니다.

## 🌟 주요 기능

### ✅ 현재 구현된 기능

- **✨ 인터랙티브 노드 관리**
  - 🎯 **스마트 자식 노드 추가**: 부모 노드 선택 시 색상과 크기 자동 상속
  - 🔄 **무한 확장 가능**: 노란색 노드에서 계속 노란색 자식 생성 가능
  - 📝 **메모 기능**: 각 노드에 상세 메모 작성 가능
  - 🖱️ **정밀한 드래그**: 5px 이상 이동 시에만 드래그 인식 (의도치 않은 이동 방지)
  - 더블클릭으로 노드 편집 모달 열기
  - 노드 크기 조절 (소/중/대)
  - 10가지 색상 팔레트로 노드 색상 커스터마이징

- **🔗 URL 링크 기능**
  - 노드에 외부 URL 직접 연결
  - 링크 클릭 시 새 창에서 열기
  - URL 제목 커스터마이징
  - 레퍼런스 및 관련 자료 관리
  - 시각적 링크 아이콘 표시

- **🎨 다양한 스타일링**
  - 원형 노드 디자인
  - 부드러운 곡선 연결선
  - 호버 효과 및 애니메이션
  - 반응형 디자인 (모바일 지원)

- **📋 템플릿 시스템**
  - 기본 아이디어 템플릿
  - 비즈니스 아이디어 맵
  - 프로젝트 계획 템플릿
  - 원클릭 템플릿 로드

- **💾 자동 저장 시스템**
  - 🔄 **실시간 자동 저장**: 모든 변경사항 즉시 LocalStorage에 저장
  - 📦 JSON 파일 형식으로 다운로드/불러오기
  - 🔁 브라우저 재방문 시 이전 작업 자동 복원
  - 💾 줌/패닝 상태도 함께 저장 및 복원
  - 📝 노드 추가, 수정, 삭제 시 자동 저장
  - 🎨 색상, 크기 변경 시 자동 저장
  - 🔗 연결선 추가 시 자동 저장
  - 파일 공유 및 백업 용이

- **🔍 줌 & 패닝 기능**
  - 🔎 **줌**: 25% ~ 300% 확대/축소
  - 🖱️ **마우스 휠 줌**: Ctrl + 휠
  - ⌨️ **키보드 단축키**: Ctrl + +/- (줌), Ctrl + 0 (리셋)
  - 🤏 **핀치 줌**: 모바일 두 손가락 제스처
  - 🖐️ **캔버스 패닝**: Space 키 + 드래그
  - 🎮 **방향 패드**: 상/하/좌/우 이동 버튼
  - ⬆️ **Shift + 방향키**: 캔버스 이동
  - 🎯 중앙 버튼으로 원래 위치 복귀

- **📱 모바일 최적화**
  - 📲 터치 제스처 완벽 지원
  - 🤏 핀치 줌 (두 손가락 확대/축소)
  - 👆 터치 드래그 (노드 이동)
  - ✋ 긴 탭으로 컨텍스트 메뉴 (500ms)
  - 📐 반응형 디자인 (모든 화면 크기)
  - 🎯 터치 영역 최적화 (최소 44px)
  - 💨 부드러운 애니메이션

- **🖼️ PNG 내보내기**
  - 📸 고해상도 PNG 다운로드 (2x scale)
  - 🎨 캔버스 전체 캡처
  - 📅 타임스탬프 파일명 자동 생성

- **⌨️ 키보드 단축키**
  - `N`: 부모 노드 선택 후 자식 노드 추가 (색상 자동 상속)

  - `Shift + 방향키`: 캔버스 이동

  - `Del`: 선택한 노드 삭제
  - `Esc`: 작업 취소/모달 닫기

  - 우클릭으로 노드 옵션 메뉴 열기

  - 노드 삭제

- **📖 프로그램 정보**
  - **프로그램 설명**: 바이브코딩 방식 소개 및 핵심 기능 안내

## 🚀 시작하기

### 접속 방법

1. 프로젝트를 로컬에서 실행하거나
2. 웹 브라우저에서 `index.html` 파일을 직접 열기

### 사용 방법

#### 1. 노드 추가 (자식 노드 확장)
- 왼쪽 도구 모음에서 **"노드 추가"** 버튼 클릭 또는 키보드 `N` 키 누르기
- **부모가 될 노드를 클릭** (예: 노란색 노드 클릭)
- 선택한 부모 노드와 **같은 색상**의 자식 노드가 자동으로 생성됨
- 부모와 자식이 **자동으로 연결선**으로 연결됨
- ESC 키로 작업 취소 가능

#### 2. 노드 편집 (메모 및 URL)
- 노드를 **더블클릭** 또는 **우클릭 → 편집** 선택
- **제목**: 노드에 표시될 제목 입력
- **메모**: 상세 설명이나 메모 작성 (50자까지 노드에 표시)
- **URL**: 관련 웹사이트 주소 입력
- **URL 제목**: 링크 설명 입력 (선택사항)
- **저장** 버튼으로 적용

#### 3. 노드 이동
- 노드를 **드래그**하여 원하는 위치로 이동 (5px 이상 이동 시 드래그 인식)
- 단순 클릭은 노드를 움직이지 않음

#### 3-1. 캔버스 확대/축소 및 이동
- **데스크톱**: 
  - Ctrl + 마우스 휠로 줌 인/아웃
  - Space 키를 누른 채 드래그로 캔버스 이동
  - 우측 하단 줌 버튼 사용
- **모바일**:
  - 두 손가락 핀치로 확대/축소
  - 우측 하단 +/- 버튼으로 줌
  - 손 버튼을 눌러 패닝 모드 활성화
- **방향 패드**: 좌측 하단 버튼으로 캔버스 이동

#### 4. URL 링크 사용
- 노드 편집 화면에서 URL을 입력하면 노드 하단에 링크 표시
- **링크 클릭** → 새 창에서 해당 웹사이트 열림
- 레퍼런스, 관련 자료, 참고 사이트 관리에 유용

#### 5. 노드 간 연결
- 노드를 **우클릭** → **"연결 시작"** 선택
- 연결할 대상 노드 클릭
- ESC 키로 연결 취소 가능

#### 6. 색상 변경
- 노드를 **우클릭** → **"색상 변경"** 선택
- 원하는 색상 클릭

#### 7. 크기 조절
- 노드를 **우클릭** → **"크기 조절"** 선택
- 소형 → 중형 → 대형 순환

#### 8. JSON 저장 및 불러오기
- 상단 **"JSON 저장"** 버튼: 
  - 현재 마인드맵을 JSON 파일로 다운로드
  - 파일명: `mindmap_[타임스탬프].json`
  - 모든 노드, 메모, URL, 연결선 정보 포함
- 상단 **"JSON 불러오기"** 버튼:
  - 저장된 JSON 파일 선택하여 불러오기
  - 이전 작업 완전 복원
- **자동 저장**: 브라우저 로컬 스토리지에도 자동 저장되어 페이지 재방문 시 이전 작업 자동 복원

#### 9. 프로그램 정보 확인
- 왼쪽 도구 모음 **"프로그램 설명"** 버튼: IdeaFlow 소개 및 기능 설명
- 왼쪽 도구 모음 **"개발 이력"** 버튼: 
  - 전체 프롬프트 히스토리 확인
  - 버전별 개선 내용 및 사용자 질의 내역
  - 향후 개발 로드맵

## 🔧 주요 기능 URIs 및 데이터 구조

### JSON 파일 구조

저장되는 JSON 파일의 상세 구조:

```json
{
  "version": "1.2.0",
  "title": "마인드맵 제목",
  "nodes": [
    {
      "id": "node-1",
      "x": 100,
      "y": 200,
      "width": 140,
      "height": 140,
      "color": "#3B82F6",
      "content": "노드 제목",
      "memo": "상세 메모 내용",
      "size": "medium",
      "url": "https://example.com",
      "urlTitle": "링크 제목"
    }
  ],
  "connections": [
    {
      "from": "node-1",
      "to": "node-2"
    }
  ],
  "zoom": 1.5,
  "panOffset": {
    "x": 100,
    "y": 50
  },
  "timestamp": 1699564800000,
  "createdAt": "2025-11-09T12:00:00.000Z"
}
```

### 필드 설명

- **version**: 파일 포맷 버전 (1.2.0)
- **title**: 마인드맵 제목
- **nodes**: 노드 배열
  - **id**: 노드 고유 ID
  - **x, y**: 노드 위치 좌표
  - **width, height**: 노드 크기
  - **color**: 노드 테두리 색상 (HEX)
  - **content**: 노드 제목 (필수)
  - **memo**: 메모 내용 (선택)
  - **size**: 크기 (small/medium/large)
  - **url**: 링크 URL (선택)
  - **urlTitle**: 링크 설명 (선택)
- **connections**: 노드 간 연결 정보
- **zoom**: 줌 레벨 (0.25 ~ 3.0)
- **panOffset**: 패닝 위치 { x, y }
- **timestamp**: Unix 타임스탬프
- **createdAt**: ISO 8601 형식 생성 시각

### JavaScript API

#### 노드 생성
```javascript
createNode({
  x: 100,              // X 좌표
  y: 200,              // Y 좌표
  width: 140,          // 너비 (px)
  height: 140,         // 높이 (px)
  color: '#3B82F6',    // 테두리 색상
  content: '내용',      // 텍스트 내용
  size: 'medium',      // 크기 (small/medium/large)
  url: 'https://...',  // 링크 URL (선택사항)
  urlTitle: '제목'      // 링크 제목 (선택사항)
});
```

#### 연결 생성
```javascript
createConnection('node-1', 'node-2');
```

## 💡 사용 팁

1. **체계적인 구조 만들기**: 중앙 노드에서 시작해 계층적으로 확장

3. **크기로 우선순위 표현**: 중요한 노드는 크게, 보조 노드는 작게
4. **링크로 레퍼런스 관리**: 관련 자료 URL을 노드에 직접 연결
5. **줌/패닝 활용**: 큰 마인드맵은 줌 아웃으로 전체 보기, 줌 인으로 세부 편집

9. **템플릿 활용**: 비슷한 구조가 필요하면 템플릿으로 시작
